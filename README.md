# 📊 Global AI & Data Science Job Market Analysis (2020–2026)

## 🔍 Overview
This project analyzes 50,000+ global AI & data science job listings to uncover trends in salaries, skills demand, and job market dynamics. It demonstrates end-to-end data analysis using Python, SQL, and Tableau.

---

## 🛠 Tools & Technologies
- Python (Pandas) – Data cleaning & preprocessing  
- PostgreSQL – Data modeling & SQL analysis  
- Tableau – Data visualization & dashboard  

---

## 📂 Dataset
- AI job listings (50,000+ records)  
- Skills demand data  
- Country-level trends  

---

## ⚙️ Process
1. Cleaned and structured raw data using Python  
2. Designed relational database in PostgreSQL  
3. Performed advanced SQL analysis (joins, aggregations)  
4. Built interactive dashboard in Tableau  

---

## 📈 Key Insights
- Experience level is the strongest driver of salary  
- Senior roles earn significantly more than mid/entry-level roles  
- Cloud platforms (AWS, Azure, GCP) dominate high-paying jobs  
- Individual skills have limited salary impact at senior level  
- Skill combinations are more important than single tools
